# Angular Labb 3 - Webbplats för konvertering

## Detta är en webbplats skapad med Angular. Syftet med uppgiften har varit att skapa en webbplats med flera undersidor samt att använda Angulars komponenter, databinding och router.

Webbplatsen innehåller en konverterare där man kan omvandla värden mellan olika enheter.

### Konverterare
Man kan konvertera följande:
Meter till feet, och vice versa
Celsius till fahrenheit, och vice versa
Kilometer i timmen till miles per hour, och vice versa

## Funktioner
- Startsida
- Om-sida som beskriver uppgiften med slutsatser
- Konverterare
- Navigationsmeny med Angular router

## Tekniner
- Angular
- TypeScript
- HTML
- CSS
- Angular forms (ngModel)
